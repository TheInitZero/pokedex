# Pokédex

A simple Pokédex application built using SvelteKit.

## Installation guide

1.  Clone the repo:

    `git clone https://github.com/TheInitZero/pokedex.git`

2.  Build the application:

    `npm run build`

3.  Preview the application:

    `npm run preview`
